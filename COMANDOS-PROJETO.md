# Scripts para Gerenciar o Projeto breezy-felix-driftfolio

## Parar o servidor (Windows)

Abra o PowerShell e execute:

```
taskkill /F /IM node.exe
```

Isso irá encerrar todos os processos node (incluindo o Vite).

---

## Iniciar o servidor

Abra o PowerShell na pasta do projeto:

```
cd breezy-felix-driftfolio
npm run dev
```

O projeto estará disponível em http://localhost:8080/
